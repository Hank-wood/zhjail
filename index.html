zhjail: 云共享你的知乎黑名单

这是什么

在知乎经常碰到三观不合的人，于是做了这个 Chrome 插件，用于与三观一致的人共享黑名单列表，一次拉黑，多人共享。

这个功能的创意来源于一个恶搞的小笑话「云把妹」——每个人将自己认识的妹子上传到云端，大家就都拥有了无数的妹子（这只是个笑话，请勿过度解读）。「zhjail」借鉴这样的思路，将优质用户的黑名单都同步到云端，所有人都可以通过 Chrome 插件将这批人一键拉黑。

由于我并不会 JavaScript，之前也从未接触过 Chrome extension 的编写，目前版本为我昨晚对着 Chrome 的文档照猫画虎做出的第一个版本，如果发生不可用敬请谅解~~~

如何使用

下载插件，由于权限原因，目前需要打开 拓展程序管理页(地址栏输入 chrome://extensions/ 打开) 进行安装。拓展程序管理页将 .crx 文件拖到 Chrome 中，如图：

点击「添加拓展程序」完成安装。

安装 zhjail 插件之后（暂时只支持 Chrome），打开你的 屏蔽设置 页（zhjail 只在这个页面被激活），如下图：

点击插件之后，会弹出一个小窗口，列出你已经封禁的用户：

点击「共享我的黑名单」，将会把你的黑名单用户上传到服务器端（为防止被人恶意将优质用户刷入黑名单，上传功能需私信我人工开启）：

点击「同步云端黑名单」，将会把服务器端的黑名单自动全部拉黑：
